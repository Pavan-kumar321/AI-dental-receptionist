# 🦷 AI Dental Receptionist

An AI-powered virtual receptionist for dental clinics that can answer phone calls, understand patient requests, and automatically manage appointments.

## Features

- 📞 AI Voice Receptionist (Vapi)
- 📅 Book Appointments
- 🔄 Reschedule Appointments
- ❌ Cancel Appointments (In Progress)
- 📆 Google Calendar Integration
- 📊 Google Sheets CRM
- 🤖 OpenAI-powered Conversation
- 🌐 English + Telugu Support
- 🔔 Appointment Reminders (Planned)
- 💬 WhatsApp Confirmation (Planned)

## Tech Stack

- Vapi
- n8n
- OpenAI
- Google Calendar API
- Google Sheets API
